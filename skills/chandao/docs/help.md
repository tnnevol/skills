# ZenTao Skill FAQ

## 常见问题

### Q: 如何配置禅道连接？
A: 设置三个环境变量：`CHANDAO_URL`、`CHANDAO_ACCOUNT`、`CHANDAO_PASSWORD`，或在 skill 目录下创建 `.env` 文件。

### Q: Token 过期了怎么办？
A: 系统会自动处理。遇到 401 时会自动重新登录，用户无需手动干预。

### Q: 支持哪些操作？
A: P0 阶段支持查询类操作：用户、产品、项目的列表和详情。后续将支持创建和状态流转。

### Q: 返回的数据安全吗？
A: 是的，所有输出都会经过脱敏处理，密码、Token、手机号、邮箱等敏感信息不会暴露。
