# 产品计划管理

| 命令 | 描述 |
|------|------|
| `/chandao productplan list --product <id> [--limit N]` | 列出产品计划 |
| `/chandao productplan list-by-product <id>` | 按产品列出计划 |
| `/chandao productplan get <id>` | 计划详情 |
| `/chandao productplan create --product <id> --title <title> [--desc <text>] [--begin <YYYY-MM-DD>] [--end <YYYY-MM-DD>]` | 创建计划 |
| `/chandao productplan update <id> [--title <title>] [--desc <text>] [--begin <YYYY-MM-DD>] [--end <YYYY-MM-DD>]` | 更新计划 |
| `/chandao productplan delete <id>` | 删除计划 |
