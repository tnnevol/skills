# 系统管理

| 命令 | 描述 |
|------|------|
| `/chandao system list --product <id>` | 列出应用系统 |
| `/chandao system get <id>` | 系统详情 |
| `/chandao system create --product <id> --name <name> [--code <code>] [--key <key>] [--desc <text>] [--type <type>]` | 创建系统 |
| `/chandao system update <id> [--name <name>] [--key <key>] [--desc <text>] [--type <type>]` | 更新系统 |
