# 项目集管理

| 命令 | 描述 |
|------|------|
| `/chandao program list [--limit N]` | 列出项目集 |
| `/chandao program get <id>` | 项目集详情 |
| `/chandao program create --name <name> --code <code> [--desc <text>] [--status wait\|doing\|suspended\|closed] [--parent <id>] [--pm <account>] [--budget <hours>] [--begin <YYYY-MM-DD>] [--end <YYYY-MM-DD>]` | 创建项目集 |
| `/chandao program update <id> [--name <name>] [--code <code>] [--desc <text>] [--status <status>] [--parent <id>] [--pm <account>] [--budget <hours>] [--begin <date>] [--end <date>]` | 更新项目集 |
| `/chandao program delete <id>` | 删除项目集 |
