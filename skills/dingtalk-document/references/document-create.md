# 文档创建

创建钉钉文档和文件夹的功能。

## 功能特性

- 创建新文档
- 创建文件夹
- 在指定知识库中创建文档

## API 调用方法

使用以下命令从 API 参考中提取详细信息：

```bash
# 创建文档
grep -A 28 "^## 6. 创建文档" references/api.md
```

## 创建流程

1. 确保必要的配置已设置
2. 获取有效的访问令牌
3. 调用创建文档 API
4. 处理返回的文档标识信息（docKey、dentryUuid、nodeId）

## 代码示例

```bash
# 创建文档后，API 会返回：
# - docKey: 用于文档正文读写的标识
# - dentryUuid: 也可用于文档正文读写的标识
# - nodeId: 用于删除和文档管理类接口的标识
```

## 关键要点

- 创建文档需要有效的知识库 ID 和操作员 ID
- 创建成功后会返回多个类型的文档标识，各有不同用途
- 文档创建后可以立即进行内容写入操作
