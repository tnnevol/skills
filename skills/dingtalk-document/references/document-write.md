# 文档写入

写入钉钉文档内容的功能，包括覆盖写入和追加写入。

## 功能特性

- 覆盖写入文档内容
- 追加文本到段落

## API 调用方法

使用以下命令从 API 参考中提取详细信息：

```bash
# 覆盖写入文档内容
grep -A 15 "^## 9. 覆盖写入文档内容" references/api.md

# 追加文本到段落
grep -A 12 "^## 10. 追加文本到段落" references/api.md
```

## 写入流程

1. 确保必要的配置已设置
2. 获取有效的访问令牌
3. 使用 docKey 或 dentryUuid 标识目标文档
4. 调用相应的写入 API

## 关键要点

- 需要使用 docKey 或 dentryUuid 来标识文档
- 覆盖写入会替换文档的全部内容
- 追加写入会在文档末尾添加新内容
- 写入操作需要相应的文档编辑权限
